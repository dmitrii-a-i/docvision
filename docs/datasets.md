## 1. MIDV-500 (116 GB)

**Путь:** `/mnt/B/Data/docvision/midv-500/`

**Описание:** 500 видеоклипов для 50 типов документов, снятых на мобильные устройства. Основной датасет для задач детекции и распознавания документов.

**Лицензия:** CC-BY-SA 2.5

**Цитирование:**
> Arlazarov VV, Bulatov K, Chernov T, Arlazarov VL. MIDV-500: a dataset for identity document analysis and recognition on mobile devices in video stream. Computer Optics 2019, 43(5): 818-824.

### Условия съёмки

| Код | Условие | Описание |
|-----|---------|----------|

### Устройства

| Код | Устройство | Формат видео |

| A | Apple iPhone 5 | .MOV |
| S | Samsung Galaxy S3 | .mp4 |

### Формат ground truth

**Разметка эталонного изображения** (поля документа):
```json
{
  "field01": {
    "value": "Sojli",
    "quad": [[334, 122], [410, 122], [410, 152], [334, 152]]
  },
  "field02": {
    "value": "Monika",
    "quad": [[334, 179], [438, 179], [438, 207], [334, 207]]
  },
  "photo": {
    "quad": [[74, 108], [301, 108], [301, 416], [74, 416]]
  },
  "signature": {
    "quad": [[64, 473], [282, 473], [282, 522], [64, 522]]
  }
}
```

- `fieldXX` — текстовые поля с UTF-8 значениями и координатами четырёхугольника
- `photo`, `signature` — специальные поля без текстового значения
- Порядок точек quad: top-left, top-right, bottom-right, bottom-left
- Документы 18 (Алжир) и 29 (Иран) — значения полей `"*"` (арабский/фарси)

**Разметка кадров** (положение документа на кадре):
```json
{
  "quad": [[334, 462], [973, 762], [758, 1179], [113, 853]]
}
```

## 2. MIDV-2019 (112 GB)

**Путь:** `/mnt/B/Data/docvision/midv-2019/`

**Описание:** Расширение MIDV-500 с двумя дополнительными условиями съёмки: сильные проективные искажения и слабое освещение. Те же 50 типов документов, но снято на более современные устройства в 4K.

**Лицензия:** CC-BY-SA 2.5

### Отличия от MIDV-500

| Параметр | MIDV-500 | MIDV-2019 |
|----------|----------|-----------|
| Условия съёмки | 5 (T, K, H, P, C) | 2 (D — искажения, L — слабый свет) |
| Видео на документ | 10 | 4 |
| Кадров на документ | 300 | 120 |
| Разрешение видео | 1920×1080 (Full HD) | 3840×2160 (4K) |
| Разрешение кадров | 1080×1920 | 2160×3840 |
| Устройства | iPhone 5, Galaxy S3 | iPhone XS Max, Galaxy S10 |

### Условия съёмки

| Код | Условие | Описание |
|-----|---------|----------|
| D | Distorted | Сильные проективные искажения (съёмка под углом) |
| L | Low-Lighting | Слабое освещение |

### Устройства

| Код | Устройство | Формат видео |
|-----|-----------|-------------|
| X | Apple iPhone XS Max | .MOV |
| G | Samsung Galaxy S10 | .mp4 |

### Структура каждого документа

```
XX_DOC_TYPE/
├── videos/              # 4 видео (2 условия × 2 устройства)
│   ├── DG01.mp4         # Distorted + Galaxy S10
│   ├── DX01.MOV         # Distorted + iPhone XS Max
│   ├── LG01.mp4         # Low-Lighting + Galaxy S10
│   └── LX01.MOV         # Low-Lighting + iPhone XS Max
├── images/
│   ├── XX_DOC_TYPE.tif  # Эталон (тот же, что в MIDV-500)
│   ├── DG/              # 30 кадров (2160×3840, ~11 МБ каждый)
│   ├── DX/
│   ├── LG/
│   └── LX/
└── ground_truth/        # Формат идентичен MIDV-500
    ├── XX_DOC_TYPE.json
    ├── DG/
    ├── DX/
    ├── LG/
    └── LX/
```

### Статистика

| Метрика | Значение |
|---------|----------|
| Типов документов | 50 |
| Видеоклипов | 200 |
| Извлечённых кадров | 6 000 (120 на документ) |
| Разрешение кадров | 2160 × 3840 (4K) |
| Размер одного кадра | ~11-12 МБ |

## 3. MIDV-2020 (200 GB)

**Путь:** `/mnt/B/Data/docvision/midv-2020/`

**Описание:** Крупнейший публичный датасет документов. 1000 уникальных документов (по 100 на тип) с уникальными значениями полей и искусственно сгенерированными лицами. 4 модальности: видеоклипы, фото, сканы, шаблоны.

**Лицензия:** CC-BY-SA 2.5 (лица сгенерированы через Generated Photos)

### Типы документов (10 шт.)

| # | Код | Описание | Код в MIDV-500 |
|---|-----|----------|----------------|
| 1 | alb_id | ID-карта Албании | 01 |
| 2 | aze_passport | Паспорт Азербайджана | 05 |
| 3 | esp_id | ID-карта Испании | 21 |
| 4 | est_id | ID-карта Эстонии | 22 |
| 5 | fin_id | ID-карта Финляндии | 24 |
| 6 | grc_passport | Паспорт Греции | 25 |
| 7 | lva_passport | Паспорт Латвии | 32 |
| 8 | rus_internalpassport | Внутренний паспорт России | 39 |
| 9 | srb_passport | Паспорт Сербии | 41 |
| 10 | svk_id | ID-карта Словакии | 42 |

### Ключевые отличия от MIDV-500/2019

- **1000 уникальных документов** с разными значениями полей (vs 1 документ на тип в MIDV-500)
- **Искусственные лица** (Generated Photos) — этически чистые данные
- **4 модальности** захвата: видео, фото, сканы (прямые + повёрнутые), шаблоны
- **Формат аннотаций:** VIA 2.0.11 (VGG Image Annotator) — отличается от MIDV-500

### Формат аннотаций (VIA 2.0.11)

**Шаблоны** — полная разметка всех полей:
```json
{
  "_via_img_metadata": {
    "00.jpg1859173": {
      "filename": "00.jpg",
      "size": 1859173,
      "regions": [
        {
          "shape_attributes": {
            "name": "rect",
            "x": 708, "y": 727,
            "width": 360, "height": 52
          },
          "region_attributes": {
            "field_name": "birth_date",
            "value": "20-11-1991",
            "features": {}
          }
        },
        {
          "shape_attributes": {
            "name": "polygon",
            "all_points_x": [0, 2167, 2167, 0],
            "all_points_y": [0, 0, 1360, 1360]
          },
          "region_attributes": {
            "field_name": "doc_quad"
          }
        }
      ]
    }
  }
}
```

**Клипы / Фото** — только face и doc_quad:
```json
{
  "_via_img_metadata": {
    "000001.jpg272797": {
      "filename": "000001.jpg",
      "regions": [
        {
          "shape_attributes": {
            "name": "rect",
            "x": 550, "y": 2245, "width": 239, "height": 300
          },
          "region_attributes": {"field_name": "face"}
        },
        {
          "shape_attributes": {
            "name": "polygon",
            "all_points_x": [360, 1830, 2070, 326],
            "all_points_y": [1996, 2085, 3019, 3096]
          },
          "region_attributes": {"field_name": "doc_quad"}
        }
      ]
    }
  }
}
```

### Поля в аннотациях шаблонов

- `photo` / `face` — регион лица
- `signature` — подпись
- `name`, `surname`, `surname_second` — ФИО
- `birth_date`, `birth_place` — дата и место рождения
- `gender`, `nationality` — пол, национальность
- `id_number`, `number` — номер документа
- `issue_date`, `expiry_date` — даты выдачи и истечения
- `authority_code` — код выдавшего органа
- `doc_quad` — четырёхугольник границ документа
